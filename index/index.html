This iframe shows your Google Doc. To make it visible to everyone, in Google Docs go to File → Share → Publish to web → Embed, then copy the iframe URL and paste it below.

If you prefer not to publish, you can try a share link with .../preview, but viewers may need to sign in and some links won't embed.

How to swap in your Google Doc

  1. Open your Google Doc → File → Share → Publish to web.
  2. Select the Embed tab → click Publish → copy the iframe code.
  3. From that code, copy just the src URL and paste it into:
    • the href on the purple “Open in Google Docs” button (top right), and
    • the src of the <iframe> above.
